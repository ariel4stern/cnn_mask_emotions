# cnn_mask_emotions
recognize mask wearing or not, if mask not found it will recognize your emotion by click.

***The models file in those google drive links for download to project:***

**emotions_model:** : https://drive.google.com/file/d/1ksGbro8L7pNO4Q5B01AwvNaMQpORTIos/view?usp=sharing  

**mask_model:**  https://drive.google.com/file/d/1AD3RSzRwBvNj1ifoEcnHF173ap3ifj25/view?usp=drive_link 


After download,
Download requirements.txt with Python 3.10.
Run streamlit_show

**For Retrain models:**

here are links to the realated zip files.

**emotions_data.zip** : www.kaggle.com/datasets/msambare/fer2013
